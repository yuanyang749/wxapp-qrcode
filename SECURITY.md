# Security Policy

## Supported Versions

Security fixes are provided for the latest version on the default branch.

## Reporting a Vulnerability

If you believe you have found a security issue in `wxapp-qrcode`, please do not open a public GitHub issue first.

Instead, contact the maintainer privately through GitHub or email with:

- A clear description of the issue
- Steps to reproduce it
- The affected usage scenario or environment
- Any suggested mitigation, if available

I will review the report as soon as possible and coordinate a fix before public disclosure when appropriate.
